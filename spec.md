# Specification

## Summary
**Goal:** Add tracking for supplier personal loans and advance payments with dashboard summaries.

**Planned changes:**
- Add new "Supplier Loans" section to record and track money borrowed by suppliers for personal use
- Add new "Advance Payments" section to track advance payments given to suppliers
- Add navigation tabs for "Supplier Loans" and "Advance Payments" to main navigation
- Update Dashboard to display total outstanding supplier loans and pending advance payments

**User-visible outcome:** Users can record and track supplier personal borrowing and advance payments, view them in dedicated sections, and see summary statistics on the dashboard.
